# DataModule + Circlefit

This is a snapshot of the data analysis library for python of the Kirchmair Group.

## Requirements

This library requires the following open source python libraries:
	- holoviews
	- bokeh
	- datashader
	- pandas
	- xarray

If you are missing some of the libaries, install it with you favourite python package managaement system (anaconda, pip, etc)

## How to use this library

Please find the instructions how to use this library in the .ipynb notebook (or .html file)
